# Timeline v3.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix canvas height clipping, default to zero selection, add monthly vertical gridlines with January emphasis, and add mouse drag panning.

**Architecture:** All four changes modify a single file (`TimelineCanvas.tsx`). The canvas height fix introduces an `AXIS_LABEL_HEIGHT` constant to break the current symmetrical padding model. Drag panning uses `useRef`-based mouse handlers on the existing scroll container. No new components or dependencies.

**Tech Stack:** Next.js 15, TypeScript, React, SVG, Vitest + React Testing Library

**Spec:** `docs/specs/2026-03-23-timeline-v3.2-design.md`

- [ ] **Step 4: Add drag handler callbacks**

After the `handleKeyDown` callback (around line 248), add:

```typescript
const handleMouseDown = useCallback((e: React.MouseEvent) => {
  const el = scrollRef.current;
  if (!el) return;
  mouseIsDown.current = true;
  isDragging.current = false;
  dragStartX.current = e.clientX;
  dragStartScrollLeft.current = el.scrollLeft;
  el.style.cursor = "grabbing";
  el.style.userSelect = "none";
}, []);

const handleMouseMove = useCallback((e: React.MouseEvent) => {
  if (!mouseIsDown.current) return; // Only drag when mouse button is held
  const el = scrollRef.current;
  if (!el) return;
  const dx = e.clientX - dragStartX.current;
  if (!isDragging.current && Math.abs(dx) < 5) return; // 5px dead zone
  isDragging.current = true;
  el.scrollLeft = dragStartScrollLeft.current - dx;
}, []);

const handleMouseUp = useCallback(() => {
  mouseIsDown.current = false;
  isDragging.current = false;
  const el = scrollRef.current;
  if (!el) return;
  el.style.cursor = "grab";
  el.style.userSelect = "";
}, []);
```
