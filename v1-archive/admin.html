Settings

Configure the agent and database connection.

This admin UI connects to your Supabase project and calls the agent via the Anthropic API. Set these in your .env file alongside agent.py:

ANTHROPIC_API_KEY=sk-ant-…
SUPABASE_URL=https://xxxx.supabase.co
SUPABASE_SERVICE_KEY=eyJ…

The admin UI reads SUPABASE_URL and the public anon key for read operations. The agent Python script uses the service key for write operations. Never expose the service key in the browser.
