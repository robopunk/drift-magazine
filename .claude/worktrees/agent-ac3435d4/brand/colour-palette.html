Typography System

Three fonts. No exceptions.
Display / Brand
Playfair Display Drift — Strategic Intelligence
Body / Editorial
Source Serif 4 The commitment has been absent from investor communications for three consecutive reporting periods. No exit statement has been issued.
Labels / Data / Code
JetBrains Mono On Record · Stage +3 · Fly · 12 signals · Mar 2026
Typography rule
Never use Inter, Roboto, or system-ui for any Drift interface. Playfair Display + Source Serif 4 + JetBrains Mono always — no exceptions, no substitutions.

Primary UI context

03

Dark UI — Text Hierarchy

Four opacity levels, neutral grey base
Parchment — Primary text (#eeeeee · 100%)
Dim — Secondary text (rgba 55%)
Faint — Labels, captions (rgba 28%)
Ghost — Disabled, decorative (rgba 12%)
Text rule
Use weight and size for emphasis, not colour jumping. Full → dim → faint → ghost. Never jump from ghost to full without context.
04

Brand Gold — The Ground Line Colour

The most significant colour in the system
commitment threshold
Fly · On Record · Run
Crawl · Drifting · Buried
Gold rule — critical
Gold is the ground line. It belongs to the commitment threshold and the brand wordmark. It is not a general highlight colour. Every use of gold carries the weight of that metaphor.

Status rule — critical
Status colours carry editorial meaning. On Record = green. Watch = amber. Drifting = orange-red. Off Record = dark red. Never repurpose these colours for decoration or other categorisation systems.
06

The Momentum Scale — Full Spectrum

Hover to expand · Sequential, never reordered
Momentum scale rule
The spectrum runs +4 (Orbit, mint) to −4 (Buried, dark red). The Watch colour (#FCD34D) is the ground line. The scale is sequential — never use these colours out of order or skip a stage.
