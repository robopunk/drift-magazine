## Editorial Standards Reference

From brand/brand-language.html, all signals must follow:
1. **Voice:** Fact-based, never sensational. Evidenced, never inferred. Precise language.
2. **Classifications:** Use exact terminology (stated, reinforced, softened, reframed, absent, achieved, morphed, transparent exit, silent drop)
3. **Confidence justification:** Each signal includes confidence score linked to evidence quality
4. **Graveyard entries:** Classified by exit manner (Silent Drop, Phased Out, Morphed, Transparent Exit)
5. **No speculation:** Absence signals require explicit evidence of language removal, not assumption
</context>
