## Firecrawl Markdown Format Reference

Firecrawl returns markdown with structure like:
```
# Company Name

## ESG Commitments
- Goal: Reduce emissions by 50% by 2030
- Target: Net-zero by 2050

### Tables (as markdown tables)
| Metric | 2023 | 2024 | Target |
|--------|------|------|--------|
| Emissions (tCO2e) | 500 | 450 | 250 |

### Dates & Metadata
Published: March 26, 2024
Last Updated: March 20, 2024
```

Signal detection should parse this to extract:
- Structured commitments (from ## sections)
- Numeric targets (from tables)
- Publication/update dates (from metadata)
- Table data for time-series analysis
</context>

   - Input: raw Firecrawl markdown string

   - content_text: plain text without markdown syntax
   - tables: list of extracted table dicts
   - timestamp: datetime or None
   - has_structured_data: bool (True if any tables or timestamp found)

4. Confidence bonus for structured data:
   - If parse_firecrawl_content shows has_structured_data=True
   - calculate_signal_confidence() adds +1 to final score (max 10)

  <action>
Implement three new functions in backend/agent.py following test expectations:

1. **extract_tables_from_markdown(markdown: str) -> list[dict]**
   - Parse markdown table blocks (identified by leading |)
   - Split table into rows, extract headers (first row after separator row)
   - Build list of dicts mapping headers to values
   - Return [] on error (no exception raising)
   - Handles: variable column counts, whitespace normalization, empty cells

2. **extract_timestamp_from_markdown(markdown: str) -> Optional[datetime]**
   - Search for patterns: "Published: ...", "Last Updated: ...", "Date: ...", "Posted: ..."
   - Parse dates using dateutil.parser.parse() with fuzzy=True (tolerant parsing)
   - Return first valid datetime found, or None
   - Handles: multiple date formats, missing dates, malformed text

3. **parse_firecrawl_content(markdown: str) -> dict**
   - Extract tables via extract_tables_from_markdown()
   - Extract timestamp via extract_timestamp_from_markdown()
   - Remove markdown syntax (##, **, **, ``, etc.) to get plain text
   - Return: {
       "content_text": str (plain text),
       "tables": list[dict] (structured data),
       "timestamp": datetime | None,
       "has_structured_data": bool (tables > 0 OR timestamp found)
     }

4. **Update calculate_signal_confidence()** from Task 1 of Plan 02-01:
   - Add parameter: has_tables: bool = False, has_timestamp: bool = False
   - Apply bonus: if (has_tables OR has_timestamp), add +1 to confidence (max 10)
   - Formula becomes: base_evidence + source_penalty + (quote_bonus + timestamp_bonus + table_bonus)

5. **Update run_monthly() and run_intake()** to call parse_firecrawl_content():
   - If Firecrawl is available and content was prefetched:
     - parsed = parse_firecrawl_content(firecrawl_content)
     - Pass parsed["has_structured_data"] to calculate_signal_confidence()
     - Attach parsed["tables"] to signal dict for future analysis (optional)
